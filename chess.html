Chess Clock

The project

This project was made over the course of a year during IGCSE as part of my Design and Technology course project.

It is a fully functional chess clock with two buttons to start and stop individual timers to simulate professional chess competition environments.

It was made for the competitive chess activity in my school, to help them practice for real championships.

The tools

The chess clock was made using 12 counter ICs and 6 ATTinys to be able to run Arduino code on them

The ATTinys were flashed using code while being connected to real Arduinos

The ATTinys and ICs were connected using custom designed and specially printed PCBS

Modelling for the PCBs was done on Autodesk Eagle and modelling for the clock was done on Autodesk Fusion
