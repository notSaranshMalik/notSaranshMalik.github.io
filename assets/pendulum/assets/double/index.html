More Information

This single pendulum system was programmed using JavaScript.

It uses general Lagrangian acceleration equations, and then a numerical approach to sample the values 60 times a second.

Due to the fact that this was made with a numerical approach, it tends to veer off the actual path around 30 seconds in, which is why the simulation pauses at that point.
