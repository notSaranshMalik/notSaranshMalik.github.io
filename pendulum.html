Pendulum Simulation

The project

This project was made while I was learning Javascript, as a way to further my interest of scientific programming while experimenting with animations in web based development.

It includes 2 graphical simulations - one for a single pendulum and one for a double pendulum. Both simulations use numerical solutions to simulate the pendulums, sampling values 60 times a second.

The single pendulum simulation aims to show the effects of air resistance, by allowing the user to modify 5 seperate variables and see how they make a difference to swing period and air resistance force.

The double pendulum simulation aims to show chaos theory, and how changing the angles and lengths of the seperate components even a small bit has a large effect on the outcome of the motion.

The tools

Javascript was used as the primary language for the simulation.

HTML and CSS were used to display the simulation and allow the changing of the variables.
