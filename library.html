Library Tracker

The project

The project was made as part of the IB Higher Level computer science course as a mini-project.

It helps keep track of a library through a single system, to manage books, users, and checkouts.

The application is cross platform, and works the exact same across Windows, Mac and Linux.

The tools

The application was made with Java, in order to keep it cross platform

The database was handled with MySQL instead of SQLite, to not need a remote database to be managed
