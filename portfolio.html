The project

This project was made in 2020 when I learnt front-end web development as a way to display my programming experience.

It includes a list of my programming projects, as well as programming experience and a way to get in contact with me.

The tools

The main site was made with HTML5, CSS3 and Javascript

No external libraries were used, and CSS3's Flexbox design patterns were used for most positioning

Javascript was used for the contact form, to send POST responses to a Google App Script to record submissions

Mobile-first design patterns were used to make the project cross-platform in the standard manner

Github Pages was used to host the files
