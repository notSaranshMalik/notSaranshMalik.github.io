Statistics Grapher

The project

This project was made for the Math department in my school as a way to teach statistical concepts. I made this project as part of my IB Computer Science IA over the course of 4 months.

It includes 2 statistical simulations - one for the T Test and one for the Chi Squared Test. In both simulations, data can be entered or imported, and the user is able to progress the simulation and show how the models are built.

The Chi Squared simulation page even has animated graphs that move and simulate the way random data is picked in that statistical test.

The tools

Python was used as the primary language for the program

The SciPy Package was used to handle the statistical parts of the simulation, using Numpy and Pandas for the data manipulation, the SciPy Library for the statiastical calculations and Matplotlib for the graphing

Tkinter was used in conjunction with the Tk Matplotlib Backend to display all the data in a windowed application

Py2App was used to freeze and bundle the application into a single file
