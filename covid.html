Covid-19 Tracker

The project

This project was made in 2020 during the Coronavirus pandemic in Singapore, as a way to track cases both regularly and by using science to measure the rate of exponentiation.

It includes 4 interactive graphs - 2 measuring the total cases till that day, as well as the new cases on that day, and the other 2 measuring the exponential nature of the spread.

The exponential graphs plot the new cases against the total number of cases, as that is the way exponential growth is modelled by differential equations. Being below any given straight line indicates the growth is no longer exponential.

The tools

The data used was pulled straight off the Government of Singapore's website, and parsed using Python and CSS modules

The data was then normalised using Numpy and Pandas to form suitable models

The graphed models themselves were written with Plotly, then embedded into an HTML / CSS based window

The data was updated daily based on the Government of Singapore's news briefings, using a Bash script
